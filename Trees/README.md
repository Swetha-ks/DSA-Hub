# Binary Search Tree Visualizer (Trees)

A Flask-based web application to demonstrate the **Binary Search Tree (BST)** data structure.  
Users can insert nodes and see the tree grow interactively, with parent-child relationships displayed visually.

---

## 🚀 Features
- Insert nodes into the BST
- Visualize the tree structure dynamically
- Show parent-child relationships with edges
- Display sorted order using inorder traversal
- Real-world analogy: file systems, organizational hierarchies

---

## 📂 Project Structure
- `app.py` → Flask backend logic
- `templates/` → HTML templates (`tree.html`)
