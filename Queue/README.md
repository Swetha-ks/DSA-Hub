# Customer Service Queue (Queues)

A Flask-based web application that demonstrates the **Queue (FIFO)** data structure using a real-world example: managing a customer service line.  
Customers join the queue, and the system serves them in the order they arrived.

---

## 🚀 Features
- Add customers to the queue (Enqueue)
- Serve customers one by one (Dequeue)
- Display the current queue in real time
- Real-world analogy: ticket counters, call centers, customer service desks

---

## 📂 Project Structure
- `queue_app.py` → Flask backend logic
- `templates/` → HTML templates (`queue.html`)
- `README.md` → Documentation
