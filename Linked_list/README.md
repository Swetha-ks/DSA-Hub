# Playlist Manager (Linked List)

A Python application that demonstrates **Linked Lists** using a real-world example: managing a music playlist.  
Each song is represented as a node in a linked list, allowing traversal forward and backward just like in actual media players.

---

## 🚀 Features
- Add songs dynamically to the playlist
- Play the current song
- Move to the **next** or **previous** song
- Display playlist status in a GUI
- Real-world analogy: Spotify/YouTube playlists internally use linked lists

---

## 📂 Project Structure
- `playlist.py` → Main Tkinter GUI + linked list logic
- `templates`   → playlist.html
