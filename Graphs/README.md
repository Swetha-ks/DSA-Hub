# Graph Visualizer(Graphs)


A Flask-based web application to demonstrate **graphs**.  
Users can add nodes and edges, and visualize connections dynamically.

---

## 🚀 Features
- Add nodes and edges
- Force-directed graph visualization
- Drag nodes to rearrange 

## 📂 Project Structure
- `graphs_app.py` → Flask backend logic
- `templates/` → HTML templates (`graph.html`)

## ⚙️ Installation
```bash
pip install -r requirements.txt
python app.py
