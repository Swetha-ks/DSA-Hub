# Stack Simulator (Flask Project)

A simple web app to demonstrate **stack operations (Push, Pop, Peek)** using Python + Flask.

## 🚀Features
- Push items into the stack
- Pop items from the stack
- Peek at the top element
- Clean Bootstrap UI

## 📂Project Structure
- Backend: Python (Flask)
- Frontend: HTML + Bootstrap

## ⚙️Run Locally
```bash
git clone https://github.com/Swetha-ks/stack-simulator.git
cd stack-simulator
pip install -r requirements.txt
python app.py
http://127.0.0.1:5000
