# Student Marks Manager (Arrays)

A Flask-based web application to demonstrate **arrays** using a real-world example: managing student marks.  
Users can enter marks, and the app calculates statistics like average, highest, lowest, median, pass/fail counts, and score distribution.

---

## 🚀 Features
- Input student marks interactively
- Display all marks neatly
- Calculate:
  - Average
  - Highest and lowest
  - Median
  - Pass/fail counts
  - Distribution of scores
- Export results to CSV

---

## 📂 Project Structure
- `array_app.py` → Flask backend logic
- `templates/` → HTML templates (`index.html`, `results.html`,`marks.html`)
